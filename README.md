Аксолотль 

Небольшая 2,5D игра вдохновленная cult of the lamb, где игрок управляет аксолотлем и должен пройти несколько комнат, где его ждут противники.

Технологии:
- язык программирования C# 12.0
- фреймворк Unity 2021.3.13f1
- дополнительно Git

Особенность проекта:
- уникальный визуальный стиль: все спрайты нарисованы мной в одной тематике
- 2.5D геймплей: 2D спрайты в 3D окружении
- атмосфера: звуковое сопровождение
